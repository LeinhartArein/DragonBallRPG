 Vegita is a Saiya-jin prince. He originally came to Earth for the 
Dragon Balls, so that he can gain immortality. However, on Namek, some 
twists make him slowly come onto the good side, although he is always 
untrustworthy. In this game, he has an M on his forward that stands for 
“Majin”, which means evil. Therefore, he is referred to as Majin 
Vegita, since Babidi possesses him.

=============
Leyenda
=============

A boton      : rodillazo
B boton      : patada fuerte
C boton      : Counter
X boton      : puño debil 
Y boton      : puño fuerte 
Z boton      : lanzar Ki

adelante      : caminar adelante
atras         : caminar adelante
arriba        : saltar
abajo         : agacharse

Start button  : saludo

=============
Basicos
=============
Dash adelante: adelante x2           (listo) 
Dash atras   : atras x2              (listo)
Cargar Ki    ; A button + X button   (50%)*falta sonido grito,efectos de carga 
Vuelo        : C button + Z button   (90%)*Faltaria un efecto de despegue y el sonido.
Agarre A     :  (Falta)
Agarre X     :  (falta)
A.I. ( inteligencia artificial): 60%
Cansancio : (90%) falta animarlo.
=============
Puños/Patadas
=============
Diagonal Kicking: atras, adelante, boton X  
                                           (90%) *Faltaria un efecto en la punta del pie
Elbow Smash: abajo, atras, boton B         (listo)

Bakuhatsu Ha: abajo, atras, boton Y

==========
Ataques de Ki
==========
Renzoku Energy Dan: presionar repetidas veces el boton C (listo).*Tip* Produce cansancio
                    cuando se acaba el ki.
KienZan:  abajo,adelante, boton A   (100%)

===============
Super Ataques de Ki
===============

Final Flash         (80%) adelante, abajo, adelante , boton B
Garlic Gun          (80%) adelante, abajo, adelante, boton Z
Explosion           (80%) atras, abajo, atras, boton x
